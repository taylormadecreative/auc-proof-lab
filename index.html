01Monday

AI as Your Team

The mental shift, taught first: stop thinking “AI is a tool” and start thinking “AI is the teammate I direct.” Live demos of research, generation, build, and analysis — with one student’s idea taken from a sentence to a live landing page while the room watches.

This is the through-line. Every day after uses what we set up here.

Day 01 output A working playbook for using Claude, Perplexity, NotebookLM, and Gemini as a co-founder, a tutor, and a research team.
Hours11 AM – 4 PM ET
FormatLecture + live demo + paired exercise
StackClaude · Perplexity · NotebookLM · Gemini

Brand & Identity

Figma fundamentals from a working designer, not a slide deck. Moodboarding, logo construction, type pairing, color theory, the difference between a brand and a logo. By mid-afternoon every student has generated their hero imagery with Gemini Flash and exported a finished brand kit.

Day 02 output Personal brand kit — logo, palette, type system, social header — exported and uploaded to their Day 01 GitHub repo.

FormatWalkthrough + studio work

§ 06 · the free stack

A free tier built like real money.

Every tool below runs in a browser, works on a Chromebook, and costs the student exactly zero dollars. The whole curriculum lives inside what’s free — because that’s the actual world freshmen are walking into.

Design
Free tier

Figma

The actual design tool working professionals use, free for students. Browser-based, real-time collaboration baked in.

Images
Free

Gemini Flash

Photorealistic image generation via Google AI Studio. Same model the pros use. Free, no credit card required.

Code
Free tier

Claude.ai

The AI coding partner that explains as it writes. Helps freshmen go from "I can't code" to "I just shipped" in one afternoon.

Hosting
Free forever

GitHub Pages

Free domain, free SSL, no server to manage. Push code, see it live. Hosts the website for as long as GitHub exists.

Mobile
Free

Expo Snack

Build React Native apps in the browser, scan a QR code, watch them run on the student's actual phone. No Mac required.
